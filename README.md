# Vadim0102/skins-advanced 📈

Пользовательский скрипт (UserScript) для сайта [csgo-skins.com](https://csgo-skins.com), который автоматически подтягивает актуальные цены со Steam Market и помогает находить выгодные предметы.

## 🚀 Функции

- **Прямое сравнение цен**: Показывает цену предмета в Steam, разницу в рублях и процентах прямо на карточке предмета.
- **Цветовая индикация выгоды**: 
  - 🟩 Зеленое свечение: предмет выгоднее на 15% и более (отличный вариант для вывода).
  - 🟥 Красное свечение: предмет переоценен на сайте на 15% и более (лучше не брать).
- **Умное кэширование**: Цены сохраняются в памяти браузера на 24 часа. Скрипт не спамит запросами и работает моментально при повторном заходе.
- **Защита от лимитов Steam (429 Error)**: Встроенная асинхронная очередь с задержками и автоматической паузой при бане по IP.
- **Быстрые ссылки**: Кнопки для быстрого перехода на Steam Market и CSFloat по одному клику.
- **Умная отмена запросов**: Если предмет пропадает с экрана (переход на другую страницу), запрос к Steam отменяется для экономии лимитов.

## 📦 Установка

Для работы скрипта потребуется расширение-менеджер скриптов для твоего браузера:

1. Установите [Tampermonkey](https://www.tampermonkey.net/) (поддерживается Chrome, Firefox, Edge, Safari).
2. Нажмите на ссылку ниже, чтобы установить скрипт:
   
   👉 **[УСТАНОВИТЬ СКРИПТ](https://raw.githubusercontent.com/Vadim0102/skins-advanced/main/script.user.js)**

   *После клика откроется окно Tampermonkey, нажмите кнопку «Установить» (Install).*
3. Готово! Перезагрузите страницу рулетки/инвентаря.
